Overview Page (index.html) – Readme

This overview (index.html) gives you three columns to manage your strategic planning:

  • Goals: Add high-level objectives. Each goal has a title, optional description, a due date, and an owner.
  • Risks: Define potential obstacles. Each risk has a title, likelihood, impact, and optional mitigation notes.
  • KPIs: Create key performance indicators. Each KPI has a name, metric type (%, $, or number), a target value, and a reporting frequency.

How to Use

  1. Create a New Item: Fill out any of the “Add” forms at the top of each column (Goals, Risks, or KPIs). Click “Save.” The new card appears immediately below.
  2. Edit or Delete: Each card has a small “×” in the top‐right corner. Click it to delete. (Editing inline is planned for MVP-3.)
  3. Drag to Timeline: Once you have at least one card in any column, go to CommandCanvas.html to drag it onto Q1–Q4 for scheduling.
  4. Persistence: All your cards and form entries are saved locally via localStorage. Refresh any page and your data reappears automatically.

Tip

Want to see how dependencies link up in the timeline? Head to CommandCanvas.html after creating your Goals/Risks/KPIs here.
